Dashboard de Vendas – Excel
Sobre o Projeto

Este projeto consiste na criação de um dashboard de vendas no Excel, com o objetivo de organizar e visualizar dados de forma clara, facilitando a análise do desempenho de vendas e a tomada de decisões baseada em dados.

Objetivo

Transformar dados brutos de vendas em informações visuais por meio de gráficos e indicadores no Excel.

Arquivos do Projeto

Dashboard_Vendas.xlsx → arquivo com o dashboard finalizado

Base_de_Dados_Vendas.xlsx → base de dados utilizada no projeto

Como Utilizar

Abra o arquivo Dashboard_Vendas.xlsx no Excel

Caso necessário, clique em Dados → Atualizar Tudo

Utilize os filtros e gráficos para analisar os dados

Ferramentas Utilizadas

Microsoft Excel

GitHub
